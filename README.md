# ✍️ Sentence Rewriter

A browser-based sentence rewriting tool built entirely with HTML, CSS, and JavaScript. Instantly rephrase, restructure, and improve your sentences — no installation or sign-up required.

***

## 📝 Overview

Sentence Rewriter is a series of 6 progressively developed HTML5 apps (`sentence rewriter 0.html` → `sentence rewriter 5.html`) that allow users to rewrite and rephrase text directly in the browser. Each version builds upon the last, introducing smarter rewriting logic, improved UI, and enhanced text-processing capabilities.

***

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- No internet connection required after download

### How to Use

1. Clone or download this repository:
   ```bash
   git clone https://github.com/endritngjelina/Sentence-Rewriter.git
   ```
2. Open the latest version in your browser:
   ```
   sentence rewriter 5.html
   ```
3. Type or paste your sentence into the input field.
4. Click **Rewrite** to generate a rephrased version of your text.

> **Tip:** Open `sentence rewriter 0.html` to see how the project evolved from a simple prototype to its current form.
